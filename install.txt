Installing Node.js on Linux/Ubuntu/ContOS
  Open Ubuntu terminal and type the following commands:
  1. sudo apt-get install python-software-properties
  2. sudo apt-add-repository ppa:chris-lea/node.js
  3. sudo apt-get update
  4. sudo apt-get install nodejs npm
  5. sudo apt-get install nodejs
The installation is now completed. You can check the version of Node by typing command:
  node --version
Checking npm version
  npm -version

Checking Installed Programs
  dpkg --get-selections
